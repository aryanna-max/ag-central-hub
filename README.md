# AG Central Hub

Sistema de gestão interna da **AG Topografia e Construções** (Aliança/PE).

Gerencia o ciclo completo de operações: captação comercial → execução de campo → processamento técnico → faturamento.

---

## Módulos

| Módulo | Função | Usuário |
|---|---|---|
| 🎯 Radar | Dashboard panorâmico — alertas, KPIs, kanban de projetos | Diretoria |
| 💼 Negócios | Leads, propostas, clientes | Sérgio, Ciro |
| 🏕️ Campo | Escalas diárias/mensais, veículos, despesas de campo | Marcelo |
| 📐 Prancheta | Processamento técnico, tarefas, kanban | Emanuel Macedo |
| 💰 Faturamento | Medições mensais, NFs, pipeline financeiro | Alcione |
| 👥 Pessoas | Funcionários, folhas de pagamento, férias | Alcione |
| 🗂️ Base | Cadastros base, usuários, configurações | Aryanna |

---

## Stack

- **Frontend:** React + TypeScript + Vite + ShadcnUI + TailwindCSS
- **Backend:** Supabase (PostgreSQL + Auth + RLS)
- **Hospedagem:** Lovable
- **Estado:** TanStack Query
